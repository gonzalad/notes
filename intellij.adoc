SHIFT + F9 : debug class (i.e. JUnit)
CTRL+N : Find class
CTRL+SHIFT+N : Find  file
CTRL+B : Go to class
F2 / Shift + F2 : fo to next / previous error
Alt+F7 : Find references (in project ?)
SHIFT + F6 : rename file
